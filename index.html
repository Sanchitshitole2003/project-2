"Stop Pollution is the Best Solution "

The Enviromnent is the surrounding of an organism. The environmnet in which an organism lives is made up of various component like air, water, soil, plastic, etc. These component are found in fixed proportions to create a harmonious balance in the environment for the organism to live in. Any kind of undesirable and unwanted changes in the proportions of these components can be termed as pollution.
